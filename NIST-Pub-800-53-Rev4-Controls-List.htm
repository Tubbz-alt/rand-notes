Controls List NIST Pub 800-53 Rev4

Security control families and controls in the NIST Special Publication 800-53 Rev4 Assessment Cases. Each has a number of separate assessment cases. The lists below identify each family and each top level Control Name within that particular family. [Reference: https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-53r4.pdf and https://nvd.nist.gov/static/feeds/xml/sp80053/rev4/800-53-controls.txt downloaded 04-09-2019] These controls may be useful in any discussion about how to engineer security into a proposed application, infrastructure, and/or services. There are 922 controls in the list below. There are 1682 numbered controls in the full list.
